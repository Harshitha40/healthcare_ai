TECH STACK:
Backend: Python, FastAPI, PaddleOCR, Groq LLM, PostgreSQL, SQLAlchemy
Frontend: React, Vite, Axios
Processing: PaddleOCR for text extraction, Groq API for LLM tasks

FEATURES:
✅ Upload PDF/Image medical records
✅ OCR text extraction with PaddleOCR
✅ LLM-based text cleaning and correction
✅ Structured medical data extraction
✅ AI-generated medical summaries
✅ Side-by-side comparison views
✅ Modern, responsive UI
✅ Real-time processing status
✅ Complete database persistence

SETUP:
1. Install PostgreSQL, Python 3.8+, Node.js 16+
2. Create database: CREATE DATABASE healthcare_ai;
3. Backend: cd backend && pip install -r requirements.txt
4. Create .env file with DATABASE_URL and GROQ_API_KEY
5. Frontend: cd frontend && npm install
6. Run backend: python main.py
7. Run frontend: npm run dev
8. Open http://localhost:3000

See README.md for detailed setup instructions.
See QUICKSTART.md for step-by-step beginner guide.
